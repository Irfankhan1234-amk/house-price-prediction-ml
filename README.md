# house-price-prediction-ml
House price prediction using data analysis, visualization, and machine learning (King County dataset)

🏠 House Price Prediction Project

In this project, I worked with the King County housing dataset to understand how different house features affect price. The goal was to explore the data, visualize important patterns, and build machine learning models to predict house prices.

📊 What I did in this project

Loaded and cleaned the dataset using Pandas

Handled missing values in important columns like bedrooms and bathrooms

Performed exploratory data analysis to understand feature distributions

Checked correlations to see which factors influence house prices the most

Built regression models (Linear Regression and Ridge Regression)

Used polynomial features and pipelines to improve prediction accuracy

Evaluated models using train-test split

📈 Observations

Houses with larger living area generally have higher prices

Waterfront homes tend to be more expensive compared to non-waterfront ones

Features like grade and location showed strong influence on price

Adding polynomial features improved model performance slightly

🛠 Tools & Libraries

Python, Pandas, NumPy, Seaborn, Matplotlib, Scikit-learn
